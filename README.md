# My Muqabala

Application mobile Flutter + backend Supabase pour le service de coaching matrimonial My Muqabala.

## Stack

- **Mobile**: Flutter 3.x, Riverpod, go_router, Material 3
- **Backend**: Supabase (PostgreSQL, Edge Functions, Auth, Storage)
- **Web**: Vercel (admin dashboard, client forms)
- **Automation**: Python scripts for data processing

  Processeur Python (processeur-global.py) 
  ------------------------------------------ 
NOTES IMPORTANTES: 
 
1. Le script utilise uniquement des bibliothèques Python standard 
2. Le script a été modifié pour forcer l'encodage UTF-8 dans la console 
3. Le script génère des codes à 6 chiffres pour les nouveaux comptes-rendus 
4. Le processus se déroule en 5 étapes: 
   - Scan des fichiers TXT 
   - Traitement des contenus 
   - Copie des assets 
   - Export CSV 
   - Configuration Vercel 
 
5. Le script crée les répertoires suivants pour chaque rapport: 
   - /output/compte-rendu/[ID]/index.html (page de login) 
   - /output/compte-rendu/[ID]/content.html (contenu du rapport) 
 
6. Le script a généré le fichier vercel.json avec routes au lieu de rewrites 
   - Nécessite correction manuelle pour Vercel
